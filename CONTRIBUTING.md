# Contributing to Napkin File Format

Thank you for your interest in the Napkin File Format proposal!

## Feedback & Discussion

We welcome feedback and suggestions via GitHub **Issues** or **Discussions**.

- **Suggest changes**: open an Issue describing your suggestion, improvement, or question.  
- **Ask questions**: use Discussions for general queries about usage, implementation, or roadmap.

Thank you for helping shape the Napkin File Format!
